# QA 전체 점검

3개의 QA 에이전트를 병렬로 실행하여 전체 품질을 점검합니다.

## 실행할 에이전트

다음 3개 에이전트를 **병렬로** 실행하세요:

1. **qa-data** (`.claude/agents/qa-data.md`): JSON 데이터 무결성 검사
2. **qa-design** (`.claude/agents/qa-design.md`): 디자인 품질 점검
3. **qa-ux** (`.claude/agents/qa-ux.md`): UX 및 사용성 점검

## 결과 보고

모든 에이전트 완료 후, 결과를 아래 형식으로 통합 요약하세요:

```
## QA 결과 요약

### 데이터 무결성
- (통과 항목 수) / (전체 항목 수) 통과
- 발견된 문제: ...

### 디자인
- (통과 항목 수) / (전체 항목 수) 통과
- 발견된 문제: ...

### UX/사용성
- (통과 항목 수) / (전체 항목 수) 통과
- 발견된 문제: ...

### 즉시 수정 필요
- ...

### 개선 권장
- ...
```

발견된 문제 중 코드 수정으로 바로 해결 가능한 것은 "즉시 수정 필요"로, 판단이 필요한 것은 "개선 권장"으로 분류하세요.
